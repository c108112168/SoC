#遊戲玩法

1.遊戲開始後，內建音樂開始撥放

2.發出音符,音符到達位置時發聲

3.當音樂發出時，按下按鈕將音符打落

4.根據打落音符的成績回報中斷信號，並於UART顯示成績

設定:於UART設定(0~3)(0為最快)
